PLG Versions

If you are using older versions of PLG there is a high chance of broken scripts, please use the newest version of PLG. Sadly, some scripts could not be recovered so there will be some missing GUIs.

  • PLG v3.2 (latest version)
  • PLG v3.1 (unsupported)
  • PLG v2.9 (unsupported)
  • PLG v2.8 (unsupported)
  • PLG v2.7 (unsupported)

More versions coming soon, and I would be willing to update some GUI designs/scripts for older GUIs depending on public demand.
